# 📸 Auto-Moment — 찍는 순간, 기록이 되다

> AI 기반 자동 타임라인 생성 및 실물 포토북 연동 서비스 소개 웹사이트  
> 발표자: 한 예 지 | 오토모먼트 기획안 기반

---

## 🎯 프로젝트 개요

**Auto-Moment**는 일상적인 사진 촬영 행위를 데이터와 결합해 자동으로 서사(Story)를 부여하는 AI 서비스입니다. 이 웹사이트는 해당 서비스의 랜딩 페이지로, 서비스 소개·프로세스·기능·시장 분석·요금제를 한 페이지에서 확인할 수 있습니다.

---

## ✅ 구현된 기능

### 📌 페이지 섹션
| 섹션 | 내용 |
|------|------|
| **Hero** | 브랜드 슬로건, 플로팅 포토카드, 통계 카운터 애니메이션 |
| **문제/해결** | 기획 배경의 문제점-해결책 시각화, 인사이트 인용구 |
| **서비스 프로세스** | 5단계 플로우 인터랙티브 카드 |
| **핵심 기능** | 자동 타임라인 UI 목업, 감정 레이어링 데모, 월간 리포트 차트 |
| **문체 설정** | 담백형/감성형/위트형 선택 + AI 생성 체험 인터랙션 |
| **시장 분석** | 시장 규모, 타겟 페르소나, 비즈니스 모델, 경쟁사 비교표 |
| **요금제** | Free/Standard/Premium, 월간↔연간 토글 |
| **사용자 후기** | 자동재생 슬라이더 |
| **CTA** | 앱 다운로드 버튼 + 사전등록 모달 |

### 🎨 디자인 특징
- 다크 테마 기반 (Navy + Purple + Pink 그라디언트)
- 스크롤 리빌 애니메이션 (IntersectionObserver)
- 카드 3D Tilt 효과 (마우스 호버)
- 파티클/오브 배경 그라디언트
- 차트 바 스크롤 진입 시 애니메이션
- 플로팅 포토카드 CSS 애니메이션

### ⚡ 인터랙션
- 키워드 입력 → AI 3가지 문체 체험 (타이핑 효과)
- 요금제 월간/연간 전환 애니메이션
- 통계 숫자 카운터 애니메이션
- 사전등록 이메일 모달 + 토스트 알림
- 모바일 햄버거 메뉴
- 스크롤 시 네비게이션 active 하이라이트
- Back to Top 버튼
- ESC 키 모달 닫기
